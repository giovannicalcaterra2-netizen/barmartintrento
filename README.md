# Bar Martin Trento — sito preview

Repository del sito statico Bar Martin Trento.

## Struttura

- `index.html`: home mobile-first con logo, eventi, foto, mappa e contatti.
- `menu.html`: menu digitale separato dalla home.
- `css/styles.css`: identità visuale e layout responsive.
- `js/content.js`: eventi e voci menu modificabili.
- `js/app.js`: menu mobile, rendering eventi/menu e animazioni leggere.

## Flusso consigliato

1. Modifica contenuti o layout.
2. Controlla la preview.
3. Pubblica solo dopo approvazione.

Nessun file deve contenere password, chiavi API o dati privati.
