# Roadmap Bar Martin AI System

## Fase 1 — Preview statica

Obiettivo: creare un sito moderno e navigabile, già pronto per essere modificato via prompt.

Output:
- landing page responsive;
- contenuti separati;
- menu e eventi modulari;
- standard social.

## Fase 2 — Repo GitHub

Creare repository `bar-martin-site` con:
- branch `main` = produzione;
- branch `preview` = anteprime;
- GitHub Pages / Netlify / Vercel.

## Fase 3 — Preview automatica

Ogni prompt genera una modifica su branch preview o pull request.
Giovanni vede il link preview e approva.

## Fase 4 — Pubblicazione controllata

Dopo approvazione esplicita:
- merge su main;
- deploy automatico;
- aggiornamento barmartin.it.

## Fase 5 — Social generator

Ogni evento produce:
- post Instagram 1080×1350;
- story 1080×1920;
- caption Instagram;
- post Facebook;
- messaggio WhatsApp;
- eventuale QR/prenotazione.

## Fase 6 — Prodotto EasySites

Trasformare Bar Martin in caso studio per vendere:
- sito statico premium;
- gestione contenuti AI;
- grafiche social;
- eventi e campagne;
- manutenzione mensile.
